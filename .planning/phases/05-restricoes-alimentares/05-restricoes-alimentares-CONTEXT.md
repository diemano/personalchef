# Phase 5 Context: Restricoes Alimentares

## Decisions
- Dietary restrictions use a binary yes/no choice first.
- If the user chooses yes, show common restriction tags and a free-text details field.
- Continue is enabled after the user explicitly answers yes or no.
- If the user chooses no, stored restrictions and notes are cleared.
