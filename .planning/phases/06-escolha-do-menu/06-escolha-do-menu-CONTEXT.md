# Phase 6 Context: Escolha do Menu

## Decisions
- Menu selection is split into four consecutive conversational screens.
- Each screen allows exactly one dish selection for its category before advancing.
- Categories are cold starter, hot starter, main course, and dessert.
- A shared component is used so dish cards and validation stay consistent.
