# Phase 3 Context: Dados do Evento

## Decisions
- Event data is split across four conversational screens: date/shift, location, guests, and occasion.
- Past dates are blocked in the calendar.
- Guest count enforces the business minimum of 10.
- Occasion can be selected from common tags or typed manually.
